2.开发环境搭建

搭建的两种方式：
1.引入.js文件来使用react
2.通过脚手架工具来编码（本文使用）脚手架工具很多，本文使用官方推荐的reactApp

create react App安装：

项目工作目录介绍：

README.md:项目说明文件
package.json:代表脚手架工具是一个node包文件，里面包含有项目相关信息，npm run start之所以运行也是因为有这个文件:

node_modules: 第三方依赖包
public/favicon.ico: 网页浏览器上的小图标
public/index.html: 首页模板

src: 源代码目录
src/index.js: 整个程序运行的入口文件
其可以引入import React是因为package.json中dependencies引入了react

src/logo.svg: 网页上旋转的那个图片

public/manifest.json: PWA serviceWorker情况下在APP桌面展示图标

网页上直接按F12可以进入开发者模式
